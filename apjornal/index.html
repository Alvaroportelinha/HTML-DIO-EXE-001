O QUE É O HTML?

(voltar ao inicio)

O HTML é uma linguagem que oference a estrutura básica de formatação de textos, imagenss, links e outros elementos, através de uma codificação chamada de "tags" que, servirão para descrever e estruturar informações dentro de um documento com do tipo de formato ".html" para criar, organizar e dar forma a nossa pagina web. Ele fornece a base para organizar o conteúdo de uma página que estamos construindo como se fosse o alicerce, o pilar, a base aonde a estrutura final do nosso site "No caso o conteudo" vai acentar.

Diferentemente de linguagens de programação, uma linguagem de marcação não executa operações ou cálculos, mas serve para representar dados ou conteúdo de forma organizada.

O termo HTML vem do ingles (HyperText Markup Language), Traduzindo - "Linguagem de Marcação de Hipertexto."

  • Hipertexto: porque tem à capacidade de criar links entre diferentes páginas ou partes do conteúdo.
  • Linguagem de Marcação porque é um sistema que usa "tags" para estruturar e organizar informações no documento.

    O QUE SÃO TAGS?

    (voltar)

    As "TAGS" são a maneira como as estruturas no html são criadas.

    Elas tem um padrão, e servem para formatar o nosso documento, dando sentido a estrutura da nossa página.

    As tags seguem um padrão de formatação abrindo e fechando com os sinais de "<" e ">" e encerrando com uma barra entre eles "/" com a informação do que ela faz.

    como por exemplo uma tag para escrever um paragrafo: <p> olá mundo!</P> aonde o resultado desta tag seriao texto dentro de aspas "olá mundo!".

    O USO DAS TAGS

    (voltar)

    Existem inúmeras tags e cada uma com a sua função, para diferentes necessidades!.

    Algumas tags servem para criar um link, outras para colocar uma imagem, outras simplesmente para criar um novo paragrafo como este aqui.

    a implementação de uma nova tag vai depender da necessidade da criação de um novo campo, formatação ou até mesmo edição.

    Como podemos ver mais abaixo alguns exemplos de utilização de algumas tags.

    TÍTULOS

    (voltar)

    A tag para titulos é escrita com a letra "h" dentro das colchetes angulares. E tal como nos processadores de texto elas são caracterizadas pela Hierarquia do título

    como por exemplo:

    TíTULO1

    TíTULO2

    TíTULO3

    TíTULO4

    TíTULO5
    TíTULO6

    não se assuste você não está fazendo um exame a vista para saber se precisa de utilizar óculos.

    esta é simplesmente a implementação da tag para títulos que na prática a a implementação seria "<h1> TÍTULO1</h1>"

    para os títulos de hierarqui de nível 1, e assim sucessivamente para os demais até a mais baixa de nível 6

    PARÁGRAFOS

    (voltar)

    como no exemplo anterior os paragrafos são escritos com as colchetes e a letra "p"

    como no exemplo acima uma tag para escrever um paragrafo: <p> olá mundo!</P> aonde o resultado desta tag seriao texto dentro de aspas "olá mundo!".

    MUDANÇA DE LINHA

    (voltar)

    as mudanças de linhas dão-se quando:

    1. estamos escrevendo um novo paragrafo
    2. estamos mudando de linha com a tag "br"
    esta tag em particular é uma das poucas que foge do padrão pois não necessita da tag de fechamento, sendo escrita no código como "<br>" e tem a função de adicionar uma nova linha ao texto.

    NEGRITO

    (voltar)

    o Negrito é escrito com a tag <strong> do inglês "forte"" e serve literalmente para fazer o que o texto diz "uma marcação com ênfase mais forte" esta tag tem um fechamento como as demais </strong>

    um explo dela em prática seria:"<strong> este texto está em negrito </strong>"

    o resultado "este texto está em negrito"

    ITÁLICO

    (voltar)

    Da mesma forma que o negrito, o itálico segue uma semantica na sua implementação

    ele é escrito com as tags <em> </em> e aqui voce já deve ter percebido que o "em" vem do "emphasis" do inglês, aondo o sentido é aplicado.

    um exemplo desta tag seria: escrita em etialico.

    mas na verdade o código por detrás dela é <em> escrita em itálico </em>

    SUBLINHADO

    (voltar)

    Nesta altura voce já deve ter percebido também que a tag "sublinhado" deverá vir do inglês como todas as outras e também deverá seguir uma semantica para fazer sentido.

    Os meus parabéns!!! você está certo.

    o código é escrito como os outros com abertura e fechamento, e a tag semantica seria o "u" do inglês "underline"

    Um exemplo de texto sublinhado: texto sublinhado

    escrito com as tags: <u> texto sublinhado</u>

    MARCADOR

    (voltar)

    Dando sequência ao próximo atributo, que neste momento você já deve ter acossiado ao inicio da nossa página aonde temos um texto marcado em amarelo mais acima.

    O "marcador" serve para dar ênfase no texto, para algumas palavras destacadas, da mesma forma que da-mos destasque, quando "grifamos" no caderno ou em algum livro que estamos a ler

    Para utilizar-mos a tag marcador basta colocar-mos as tags delimitadoras <mark> texto marcado</mark>

    resultado: texto marcado

    PEQUENO

    (voltar)

    A tag "small" serve para escrevermos um texto mais pequeno, o que dá muito jeito na criação de páginas, porque com alguma criatividade podemos ter uma aplicação muito ampla deste, anotações, textos de rodapé, ou até mais criativos como textos com links para voltar, como as tags que estãos escritas abaixo de cada título que ao clicar volta para o nosso índice

    a sua aplicação é feita com as tags <samll> um exemplo de texto pequeno</small>

    resultado: um exemplo de texto pequeno

    LISTAS

    (voltar)

    As listas são umas das tags que normalmente mais vamos usar, porque:

    1. são muito úteis
    2. acrescentam organização e estética a nossa pagina

    a criação de listas é feita com a tag <li> e o conteudo que ficará dentro da lista</li>

    entretanto, devemos ter em conta pontos cruciais como:
    • A ordem
    • A semantica

    como pode reparar mais acima existem dois tipos de listas ordenadas e não ordenadas

    LISTAS ORDENADAS

    (voltar)

    As listas ordenadas, são criadas de forma sequêncial tendo em conta a "ordem"

    a tag para a sua criação é a tag <ol> </ol> do inglês "ordenadat list" e a asua implementação é feita com a tag <li></li> dentro dela exemplo: <ol> <li> </li> </ol>

    exemplo:

    <ol>
    <li> o conteudo da lista ficará aqui</li>
    <li> o outro ficará aqui</li>
    <li> e o outro ficará aqui</li>
    </ol>

    1. o conteudo da lista ficará aqui
    2. o outro ficará aqui
    3. e o outro ficará aqui

    LISTAS NÃO ORDENADAS

    (voltar)

    Da mesma maneira as listas "não ordenadas vêm do inglês e neste caso do inglês "undernadate list" ou "lista não ordenada"

    a sua aplicação é feita com as colchetes em "ul"

    a sua implementação é feita com a tag "li" dentro dela da mesma forma que as das listas ordenadas

    exemplo:

    <ul>
    <li> uma lista não ordenada</li>
    <li> o com muitos campos</li>
    <li> que servem para exemplo</li>
    </ul>

    • uma lista não ordenada
    • com muitos campos
    • que servem para exemplo

    LINHA ESTÉTICA

    (voltar)

    Uma linha estética é um separador que dá um loyalt mais bonito a página, não é nada mais do que uma linha na página

    ela é escrita com a tag "hr" e tal como a tag "br" não tem fechamento.

    Exemplo: <hr>

    INFERIOR

    (voltar)

    A tag inferior é escrita com as letras "sub" dentro de colchetes.

    ela é muito útil para escrever fórmulas por exemplo ou qualquer outra palavra que precise ficar abaixo da sequencial

    Exemplo:

    <p>H<sub>2</sub>O</p>

    H2O

    você consegue reparar no 2 abaixo da linha dos demais?

    é muito útil essa tag para nos permitir ser-mos mais flexíveis com a escrita

    SUPERIOR

    (voltar)

    para escrevermos o superior só precisamos seguir a mesma logica

    se o "sub" é para baixo o "sup" é para cima e tem muita utilidade para escrevermos formulas matemáticas por exemplo

    <p>10<sup>2</sup>=100</p>

    102=100

    a leitura é muito clara,e isso ajuda na interpretação do texto.

    LINKS

    (voltar)

    Os links são um dos pontos chaves e mais fortes "se não o mais" da linguagem HTML

    pois nos permitem "navegar" entre pontos do documento ou pagina, ou entre páginas

    a tag de link é escrita com a letra "a" dentro das colchetes e o seu caminho é colocado dentro da referência entre as aspas que ficaria como: href=""

    um exemplo seria <a href=""> </a>

    <a para a abertura do link
    href="" para o caminho que ele vai percorrer

    </a> para fechar o link

    exemplo:<a href="#home"> este link vai para o íncide </a>

    este link vai para o índice

    ID

    (voltar)

    Dando sequência, vamos falar sobre os ids.

    Lembra que as ancoras são as portas de entradas? pois é elas precisam de um porta de saída, e os ids nos dão esta solução.

    pode-mos escrever os ids com as letras "id", e como os ids são atributos elas seguem a mesma logica "ficando dentro de uma tag"

    Se colocarmos uma porta de entrada "uma ancora" ela vai precisar de uma porta de saída "um id"

    que dentro de uma tag ele ficaria assim:

    <h1 id="home"> índice </h1>

    a essa altura você já sabe que o h1 é uma tag de titulo e que o id dentro dela é um atributo
